Unit tests are coded tests that test the functionality of other code. You can think of them as the handcalc of software engineering.

Google "Unit Tests" for more information

The unit test coverage for these macros is really weak, but it's improving. 

Inorder to run all the unit tests for the ANSYS_MACROS, double click RunAll.bat, check OUT_UNIT.out for the results of the unit tests.
